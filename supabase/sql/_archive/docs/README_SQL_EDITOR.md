# Supabase SQL Editor - النسخة النظيفة V29

للتشغيل اليومي لا تستخدم ملفات Patch القديمة.

## 1) شغّل هذا الملف فقط داخل Supabase SQL Editor

```text
supabase/sql/RUN_IN_SUPABASE_SQL_EDITOR.sql
```

## 2) بعد نشر Edge Functions شغّل ملف التحقق

```text
supabase/sql/VERIFY_AFTER_SUPABASE_DEPLOY.sql
```

الملفات القديمة موجودة للرجوع فقط داخل:

```text
supabase/sql/_archive/
```

لا تشغّل ملفات الأرشيف إلا لو كنت ترجع لإصدار قديم عن قصد.
