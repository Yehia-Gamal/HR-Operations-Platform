# تقرير تنفيذ ملاحظات الموظفين — 2026-05-04

تم تطبيق دفعة الملاحظات التي أرسلها المستخدم على نسخة Neon Command Center Theme V3 + Identity Guard V4.

## البنود المنفذة

1. **عرض الموقع بشكل مفهوم**
   - إضافة عرض نصي: `مجمع أحلى شباب — منيل شيحة - الجيزة` بدل الإحداثيات وحدها.
   - إظهار حالة داخل/خارج المجمع بشكل Badge واضح.
   - الاحتفاظ بالدقة والمسافة كمعلومات ثانوية فقط.

2. **هيدر الموظف**
   - صورة + اسم + مسمى وظيفي.
   - تكبير اللوجو وتحسين اتزان الهيدر.

3. **إعادة ترتيب الرئيسية**
   - الترحيب والحالة اليومية.
   - كارت البصمة.
   - كارت الموقع.
   - إجراءات سريعة موحدة.
   - حالة اليوم.
   - آخر البصمات والطلبات في الأسفل.

4. **صفحة البصمة**
   - تبسيط بطاقة الفرع.
   - نقل تحديث بصمة الجهاز إلى حسابي/أمان الجهاز كمفهوم، وعدم إظهاره كزر رئيسي بجوار البصمة.
   - حقل ملاحظة مرتبط بالبصمة نفسها.
   - تمييز حضور/انصراف حسب توقيت اليوم وحالة اليوم.
   - زر صغير لاختبار الموقع وعرض خريطة Google.
   - حذف كلمة `web` من العرض المرئي.

5. **الإشعارات والسياسات**
   - Toast مؤقت يختفي بعد 5 ثوانٍ.
   - عدم عرض رسائل النجاح كلوحات ثابتة داخل الصفحة.
   - زر تأكيد السياسات يستخدم endpoint موجود ويعرض Toast بالنتيجة.

6. **الإجازات والمأموريات**
   - مسار طلبات بمرحلتين: المدير المباشر ثم HR.
   - إضافة حالات workflow داخل payload.
   - إضافة صفحة `فريقي` للمدير المباشر.

7. **KPI Workflow**
   - تحويل البنود إلى sliders من 0 إلى 100.
   - حساب الدرجة المحتسبة بناءً على الوزن.
   - توضيح مراحل الموظف/HR/المدير/السكرتير/الأرشفة.
   - إضافة Patch 058 لدعم الأعمدة والحساب.

8. **الشكاوى والخلافات**
   - اختيار طرف معين أو شكوى عامة.
   - Checkboxes للتكرار العام والتكرار مع نفس الشخص.
   - خصوصية اللجنة دون مرور على المدير المباشر.
   - Patch 059 لمسار لجنة الخلافات والمحاضر والتصعيد.
