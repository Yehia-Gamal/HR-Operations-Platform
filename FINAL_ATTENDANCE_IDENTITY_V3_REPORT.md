# تقرير تطبيق Attendance Identity Guard V3

تم تطبيق حزمة V3 لمنع تلاعب الحضور عبر تبادل الموبايلات أو الحسابات.

## تم التنفيذ

- إقرار سياسة حضور إلزامي عند أول بصمة.
- جهاز موثوق يحتاج اعتماد HR.
- QR متغير داخل الفرع.
- Anti-GPS spoofing signals.
- Browser install id.
- حضور احتياطي عند فشل الكاميرا/GPS/Passkey.
- SQL Patches 053–056.
- مركز مخاطر وتقرير شهري.
- تحديث فحص `check:attendance-identity` للتحقق من مكونات V3.
