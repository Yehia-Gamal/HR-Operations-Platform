# الهيكل الإداري — نسخة بيانات عامة

تم تنظيف بيانات الهيكل الإداري داخل الحزمة بحيث لا تحتوي على أسماء شخصية حقيقية. استخدم هذه البيانات كقالب، ثم أدخل أسماء الموظفين الحقيقية من لوحة الإدارة أو من ملف استيراد رسمي بعد النشر.

## ملاحظات تنفيذية

- الأرقام والإيميلات داخل الحزمة افتراضية وليست بيانات اتصال حقيقية.
- إنشاء حسابات الدخول الحقيقية يتم من لوحة الإدارة أو Edge Functions، وليس من SQL مباشر.
- يمكن تعديل الهيكل من لوحة الإدارة بعد تشغيل Supabase الحقيقي.
