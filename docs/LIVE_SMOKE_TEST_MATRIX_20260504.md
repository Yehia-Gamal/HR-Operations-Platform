# مصفوفة اختبار التشغيل الحي

| المجال | الاختبار | المتوقع |
|---|---|---|
| Login | دخول موظف برقم الهاتف | يعمل بدون كشف بيانات حساسة |
| Header | ظهور صورة + اسم + وظيفة | يظهر بوضوح على الموبايل |
| Location | إرسال الموقع | يظهر عنوان مفهوم + حالة داخل/خارج المجمع |
| Attendance | بصمة حضور داخل المجمع | تسجل داخل المجمع بعلامة خضراء |
| Attendance | بصمة خارج المجمع | تحتاج ملاحظة/مراجعة حسب السياسة |
| Device | جهاز جديد | يظهر طلب اعتماد جهاز |
| Shared Device | موظفان من نفس الجهاز | يظهر SERVER_SHARED_DEVICE_RECENT |
| QR | بصمة بدون QR فرع | تُرفع درجة الخطر أو تتحول لمراجعة |
| Leave | طلب إجازة | مدير مباشر ثم HR |
| Mission | طلب مأمورية | مدير مباشر ثم HR |
| My Team | صفحة فريقي | تظهر للمدير المباشر فقط |
| KPI | دورة مفتوحة للموظف | sliders 0–100 ويحسب الوزن تلقائيًا |
| KPI | HR stage | HR يملأ بنود HR فقط |
| KPI | Manager stage | المدير يعدل غير بنود HR |
| Disputes | شكوى ضد شخص | اللجنة ترى التفاصيل، الطرف يرى إشعارًا عامًا فقط |
| Policies | تأكيد قراءة السياسة | يتم حفظ التأكيد وتغيير الحالة |
| Toast | أي رسالة نجاح/خطأ | تختفي تلقائيًا بعد 5 ثوانٍ |
| Theme | الثيم الجديد | يظهر في كل البوابات |
| Offline | offline.html | يظهر عند انقطاع الشبكة |
