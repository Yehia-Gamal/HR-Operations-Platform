# Mobile Executive + Push Fix Report

## تم التنفيذ

- كود بوابة التشغيل الحالي في النسخة التجريبية: `LOCAL-DEMO-GATE`.
- تحسين عرض كروت صفحة المتابعة التنفيذية على الموبايل ومنع تداخل الاسم/الصورة/الحالة.
- إصلاح ظهور القائمة الجانبية في صفحة `/executive/` وعدم إخفائها بعد الضغط على زر القائمة.
- إضافة Fallback تلقائي لصور الموظفين من `shared/images/employees` عند عدم وجود `photoUrl` محفوظ.
- عند كسر رابط الصورة يظهر اختصار الاسم بدل أيقونة صورة مكسورة.
- تغيير حذف الموظف في الوضع المحلي للمستخدمين التنفيذيين إلى حذف نهائي من بيانات التشغيل المحلية، مع تنظيف الحساب والسجلات المرتبطة.
- تحسين حفظ كلمة المرور المؤقتة من شاشة المستخدمين وإضافتها إلى خزنة كلمات المرور المؤقتة مع إشعار للموظف.
- تحسين دعم إشعارات الموبايل/PWA:
  - زر تفعيل الإشعارات داخل تطبيق الموظف.
  - إشعار متفاعل عند طلب الموقع أو وصول تنبيه غير مقروء.
  - Service Worker يدعم فتح الصفحة المناسبة عند الضغط على الإشعار.
  - اهتزاز وتنبيه نظام عند دعم المتصفح.

## ملاحظات مهمة

إشعارات نظام التشغيل خارج التطبيق تحتاج HTTPS، وتثبيت/السماح بإشعارات الموقع على الموبايل. الإشعارات الحقيقية بين أجهزة مختلفة تحتاج Supabase Live/Push أو Web Push server/VAPID؛ هذه الحزمة جهزت الواجهة والـ Service Worker وسجلات الاشتراك، لكن الإرسال العابر للأجهزة يعتمد على إعداد الباك إند.
