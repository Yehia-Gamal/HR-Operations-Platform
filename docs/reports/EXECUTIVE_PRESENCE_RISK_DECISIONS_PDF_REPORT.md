# تقرير تطبيق إضافات المدير التنفيذي والحضور اللحظي — v1.3.4

تم تطبيق حزمة تعديلات تشغيلية على نسخة HR Supabase:

## الميزات المضافة

1. **لوحة حضور لحظية للمدير التنفيذي**
   - صفحة `presence-map` داخل لوحة الإدارة.
   - تبويب `خريطة الحضور` داخل بوابة المدير التنفيذي.
   - عرض الحاضر، المتأخر، الغائب، من لم يرسل الموقع، وخارج النطاق.
   - روابط Google Maps لكل موظف لديه GPS محفوظ.

2. **نظام تقييم خطر البصمة**
   - صفحة `attendance-risk` داخل لوحة الإدارة.
   - تبويب `مخاطر البصمة` داخل بوابة المدير التنفيذي.
   - رصد: تكرار بصمة سريع، موقع خارج النطاق، جهاز جديد، مسافة بعيدة، وبصمة بدون GPS.

3. **سجل قرارات إدارية رسمي**
   - صفحة `admin-decisions` داخل لوحة الإدارة.
   - تبويب `قرارات` داخل بوابة المدير التنفيذي.
   - صفحة `القرارات` داخل تطبيق الموظف.
   - كل موظف يضغط “تم الاطلاع” ويتم حفظ توقيت القراءة.

4. **محاضر لجنة حل المشاكل والخلافات**
   - إضافة زر “محضر لجنة” في صفحة لجنة الخلافات التنفيذية.
   - حفظ القرار، الملاحظات، أعضاء اللجنة، وتوقيع المستخدم الحافظ للمحضر.

5. **صلاحيات المدير المباشر**
   - Patch 043 يضيف دالة `current_can_see_employee` لتمكين سياسة “المدير يرى فريقه فقط”.
   - إضافة صلاحية `manager:team-only`.

6. **تقارير PDF شهرية تلقائية**
   - صفحة `monthly-auto-pdf` داخل لوحة الإدارة.
   - تجمع الحضور، التأخير، الغياب، KPI، الشكاوى، والطلبات.
   - جاهزة للطباعة أو الحفظ PDF من المتصفح.
