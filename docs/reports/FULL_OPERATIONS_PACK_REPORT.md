# تقرير حزمة التشغيل الكامل — 020

تم تطبيق حزمة تطوير واسعة على النظام لتغطية التعديلات المقترحة الـ 26 قدر الإمكان داخل نسخة الويب الحالية.

## صفحات وإمكانيات الإدارة الجديدة

- تقرير تنفيذي مختصر للشيخ محمد يعرض: الموظفين، حضور اليوم، الطلبات المعلقة، المشاكل المفتوحة، المهام المتأخرة، والمستندات قرب الانتهاء.
- صفحة أرصدة الإجازات لإدارة الرصيد السنوي والعارض والمرضي والمستخدم والمتبقي.
- صفحة المهام الداخلية لإنشاء مهمة لموظف، تحديد أولوية وتاريخ استحقاق، وإغلاق المهمة عند الإنجاز.
- صفحة مستندات الموظفين لحفظ بيانات الملفات وروابطها وتاريخ انتهاء المستند.
- صفحة مصفوفة الصلاحيات لتعديل صلاحيات الأدوار من الواجهة.

## تطبيق الموظفين

- إضافة صفحة “طلباتي” لتجميع الإجازات والمأموريات وطلبات الموقع والاستثناءات.
- إضافة صفحة “مهامي” ليشاهد الموظف التكليفات ويغلقها عند الانتهاء.
- إضافة صفحة “مستنداتي” ليعرف الموظف الملفات المسجلة والتنبيهات الخاصة بها.
- إضافة اختصارات جديدة للمهام والمستندات من الصفحة الرئيسية.

## Supabase

تمت إضافة Patch جديد:

```text
supabase/sql/patches/020_full_operations_pack.sql
```

ويضيف الجداول التالية:

- `leave_balances`
- `employee_tasks`
- `employee_documents`
- `announcement_reads`
- `policy_acknowledgements`
- `v_employee_operations_summary`

كما يضيف صلاحيات جديدة مثل:

- `tasks:manage`
- `documents:manage`
- `leave:balance`
- `executive:report`
- `permissions:matrix`
