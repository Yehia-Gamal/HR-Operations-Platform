# Neon Command Center Theme V3 Guide

هذا الدليل يوضح طبقة V3 من الثيم الداكن المطبق على نظام HR.

## ما أضافته V3

- تحسين وضوح الـ focus عند استخدام لوحة المفاتيح.
- توحيد أحجام أزرار اللمس للموبايل.
- تمييز بصري واضح لحالات المخاطر العالية والمتوسطة والمنخفضة.
- تحسين الجداول العريضة على الموبايل.
- دعم `prefers-reduced-motion` لتقليل الحركة لمن يحتاج ذلك.
- تحسين Empty States ولوحات المساعدة.

## متغيرات مهمة

- `--neon-focus`
- `--neon-danger-glow`
- `--neon-warning-glow`
- `--neon-ok-glow`

## قواعد الاستخدام

استخدم هذه الكلاسات للحالات:

- `risk-high` أو `data-risk="high"`
- `risk-medium` أو `data-risk="medium"`
- `risk-low` أو `data-risk="low"`

كلها مربوطة مباشرة بستايل V3.
