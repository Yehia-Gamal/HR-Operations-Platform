# اختبار القبول النهائي للويب

## الأدمن

- تسجيل الدخول.
- فتح لوحة الأدمن من بوابة التشغيل فقط.
- إضافة موظف.
- تعديل موظف.
- تغيير كلمة مرور مستخدم.
- رفع صورة موظف.
- طلب موقع مباشر.
- مراجعة الحضور والطلبات.
- التأكد من ظهور صفحات النظام التقنية للأدمن فقط.

## المدير التنفيذي

- فتح المتابعة التنفيذية من بوابة التشغيل.
- التأكد أن الواجهة مختلفة عن الأدمن.
- التأكد من عدم ظهور Supabase / Database / Backup / Roles.
- طلب موقع مباشر.
- فتح تفاصيل موظف.
- مراجعة مؤشرات الغياب والتأخير.
- محاولة فتح `/admin/` يجب أن تعيد إلى البوابة ولا تفتح الأدمن بجلسة التنفيذي.

## الموظف

- تسجيل دخول.
- حضور وانصراف.
- إرسال موقع عند الطلب.
- طلب إجازة/مأمورية.
- رفع شكوى.
- تغيير كلمة مرور.
- تعديل بيانات الاتصال.

## الموبايل

- Android Chrome.
- iPhone Safari عند توفره.
- شاشة صغيرة 360px.
- شاشة متوسطة.
- Dark Mode.
- تثبيت PWA من المتصفح.
- تجربة انقطاع الإنترنت والعودة.
